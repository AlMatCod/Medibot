# Medibot
**Medi** — это Telegram-бот, который помогает пользователю с домашними советами по симптомам.  

⚠️ Важно: бот даёт **только советы**, не ставит диагноз и не заменяет врача.

## Как работает

- Пользователь пишет боту свои симптомы, например: `Болит голова` или `Температура 38`.
- Бот пишет `Подождите, Medi думает...` и потом отвечает безопасным советом.
- Используется бесплатный API [ApiFreeLLM](https://apifreellm.com/) для генерации ответов.

## Команды

- `/start` — приветствие и инструкция по использованию.

## Пример использования


/start
Привет! Я Medi, твой медицинский помощник. Опиши свои симптомы, и я дам советы, как действовать.
Болит голова и тошнит
Подождите, Medi думает...
Попробуй отдохнуть, пить воду, проветрить комнату. Если боль не проходит — обратись к врачу.


## Установка

1. Установите Python 3.10+
2. Скачайте проект и установите библиотеки из `requirements.txt`
3. Запустите `bot.py`
